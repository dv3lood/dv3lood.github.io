Created2021-06-14|Updated2021-07-06
|Post View:

ADT(Abstract Data Type)，全名抽象数据类型。对于一个抽象数据类型，我们仅仅关心其具有的操作，并不关注其具体的实现方式和存储方式。

其中抽象数据类型的操作可以按照以下进行分类：

  • Creators: 产生新类型的对象

    • t* -> T

  • Producers：在已有对象基础上产生新的对象

    • T+, t* -> T

  • Observers：输入本抽象类型，返回其他类型

    • T+, t* -> t

  • Mutators：改变本对象

    • T+, t* -> void|t| T

AF(Abstraction Function),它将ADT与现实空间中的某个对象相关联，AF就是描述这种关联具体是什么样的；

RI(Representation invariant),指的是每个变量的现实意义和这个ADT中的变量需要遵守怎样的准则，也就是判断其内容是否合法。
